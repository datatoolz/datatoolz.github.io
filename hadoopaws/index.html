Setting up Hadoop Cluster on AWS

 In this tutorial we will be setting up a 4 node Hadoop cluster on the cloud. For a 4 node cluster we need 4 machines right? Amazon has a free tier service which we can utilize for this purpose. We can create 4 amazon EC2 instances each serving as a node. An EC2 instance is a virtual server in Amazon's Elastic Compute Cloud (EC2) for running applications on the Amazon Web Services (AWS) infrastructure. In short an EC2 instance is nothing but a virtual computer.

 I will be explaining the step by step process to setup a 4 node Hadoop cluster on AWS EC2. You need to have an amazon web service account to begin with. You can create an account here - https://aws.amazon.com/

Once you have created an account sign in to the console. Once logged in you will see a screen like below;

 

From there click on Services and EC2. Then click on the Launch Instance button. The steps are very intuitive and I will put a gif image of the process.
